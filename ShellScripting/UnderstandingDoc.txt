shell: linux provides a special interpreter program which can be used to execute commands of the opreraing system.

Directory Structure:
====================
/bin - all executable files
/boot - boot info
/dev - device files, dvd ,cd etc..
/etc - all configuration files
/home - user Directory paths
lib, lib64 - library files.

/opt - third party software files like chrome

Commands:
=========

touch - create empty files
mkdir - Create Directory
cat - to see file content and create files
ls - list files and directories.
cd - change Directory
pwd - present working Directory
date- to see date and change system date
cal - to see calender

. - represents current Directory
.. - represents previous directory
